This repository is maintained by Iress and is **private**. While the resulting package may be published to [npm](https://www.npmjs.com/), the source code is not open to external contributors.

### External Reporting

Please report all security concerns to your Iress contact person.

Include as much detail as possible, such as:

- A description of the vulnerability
- The impact and severity
- Steps to reproduce
- Any relevant logs, screenshots, or code snippets
