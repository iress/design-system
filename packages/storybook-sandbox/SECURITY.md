# Security Policy

This repository is maintained by Iress and is **public**. The package is published to [npm](https://www.npmjs.com/) and the source code is open to external contributors.

If you believe you have discovered a security vulnerability within this component library or its published package:

- **Do not open a public GitHub issue or pull request** that discloses the vulnerability.
- **Instead, please report security vulnerabilities privately** using one of the following methods:

### Reporting Methods

1. **GitHub Security Advisories** (Preferred): Report via the [Security tab](../../security/advisories/new) in this repository
2. **Email**: Contact the Iress Design System maintainers directly
3. **Iress Employees**: Report to your internal security team or the Frontend Enablement team

### What to Include

Please provide as much detail as possible:

- A description of the vulnerability
- The impact and severity
- Steps to reproduce
- Any relevant logs, screenshots, or code snippets
- Suggested remediation (if available)
